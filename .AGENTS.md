# Autonomous AI Agents Guidelines

This repository supports integration with autonomous AI agents (such as Antigravity).
When AI agents are contributing to this codebase, they should adhere to the following rules:
- Respect the "Local First" philosophy of the project.
- Write tests for new view models.
- Maintain the Fluent Design System aesthetics (Mica, WinUI 3 controls).
- Update `.md` documentation files for any architectural changes.
