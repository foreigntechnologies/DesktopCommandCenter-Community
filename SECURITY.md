# Security Policy

## Supported Versions

Only the current major version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.0.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please DO NOT report it publicly. Instead, send an email to the development team. All security vulnerabilities will be promptly addressed.
